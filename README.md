# sparkl

Site for verified independent cleaners.

## Run locally

Because this is a static site, you can run it with any local file server:

```bash
python3 -m http.server 8080
```

Then open: `http://localhost:8080`.

## What's implemented

- Browse cleaner cards
- Filter by service tag
- Search by neighbourhood, cleaner name, or service keyword
- Sort by price, rating, and review count
- Cleaner profile view
- Booking request modal flow
- Cleaner application form success state
